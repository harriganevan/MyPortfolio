Trip Planner

Trip Planner is a web application that allows users to create and plan trips. The application provides a geographic map and allows users to search for nearby attractions of their destinations. Users can create an account and save their trips. Accomplished using React, Express, Node.js, and MongoDB.

Graphical Password

Graphical Password is a web application login system that uses images selected in a specific order as a user's password. The application features robust accessibility features for a memory impaired and neurodiverse demographic. Accomplished using Flask/Python, JavaScript, HTML/CSS. Regarded as 'best implementation' of this project by professor.

Lumen

lumen uses a circuit constructed with photoresistors connected to a Raspberry Pi running a Python script to determine how much light is directed towards the circuit. The data is stored in a SQL database and can be graphed.
